# Handy Media Python Scripts

A modern indie-dev showcase website featuring simple, powerful Python utilities for working with subtitles and media files.

---

## Featured Scripts

### 1. SRT Translator
Translate subtitle files while preserving all timestamps. Supports LibreTranslate, Google Translate, and DeepL.

### 2. Dual-Movie Player
Play two videos simultaneously with instant keyboard switching. Perfect for language learning.

### 3. SRT Transcript Converter
Convert subtitle files into clean, readable transcripts with periodic time markers.
